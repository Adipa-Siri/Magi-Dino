# CMP105 Alternative Project

## Game 

**Dino Hanyman: Spanner in the Works** 

A two-level platformer prototype developed as an alternative trunk project to extend for CMP105 students from 2026 onwards. This project has most of the base requirements for a group submission. In the first level, the player has to navigate to the far right-side of the screen, avoiding pitfalls, and activate a lever before returning to the start. In the second level the player must investigate the high wall, backtrack to a box and jump into it from below to reveal a coin, colliding with the coin grants the ability to double-jump, using which they can navigate to the end of the level.

The engineering for this game is restricted to solely using the framework with few additions. 

**Controls:** 

WASD to move,
Spacebar to jump,
F to interact (prompts will appear when possible),
R to reset (if you get stuck or fall down)
P to pause,
L-Control to dash,
left mouseclick in the menu.

# Proposal

## Brief Overview of game extention 

Dino Handyman: A dino who needed to find a flower to trun into a magical girl. The game will be two levels side scrolling game that the player cold throw projectile at the enemy. The first level focus on hitting enemies and jumping on to a platform to get to the flower. The second level the player will enter a boss fight where the player needed to dodge the projectile as well as inflicting damage to defeat the boss.

## Must Have Features

* Projectile Throwing: The player will be able to right click to throw a projectlie and the enemy could summon projectile
* Pausing Screen: Pressing esc to pause the game and go back to the menu
* Enemies: Enemies should follow the player and has a gap check between platforms

## Should Have Features

* Data-driven Level Loading: To create a level large enough to introduce the new mechanics without making the code a mess. 
* Health: The player will have 20 health to take damage from certain enemies
* Enemy drop item: Killing enemies will randomly give player thier projectile or healing item.

## Could Have Features

* Dialouge system: Player could interact with NPC for explaining what the game's about
* Power-ups:Player gain power up by claiming enemy's drop making the attack stronger for a while 

## Wish to Have Features

* Pay to win system: By paying the player will get 3 times the hp and the player projectile will be an instant kill. 

* Pet system: The player can select a pet before going into the level and have them as a partner helping the player complete the level easily.
